Mise En Place

Mise En Place is a responsive web-based cooking app. This was created using Figma, Adobe Illustrator, and Google Slides.

This app features user research, personas, and interviews, visual design style guide, user flows, rapid prototyping, mid and high fidelity wireframes, responsive breakpoints, prototyping, and mockups.
